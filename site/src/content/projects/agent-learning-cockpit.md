这个项目把学习从“收藏链接”推进到“形成可复用判断”的阶段。它不是知识库皮肤,而是一个帮助 agent 和人分工的工作台:agent 负责收集、抽取、比对和初筛,人保留真正的判断、取舍和验收。

当前重点是把阅读材料、研究问题、验证任务和发布草稿连接起来,让每一次学习都留下可复用的结构,而不是只留下零散笔记。
