# 这里是测试

## 什么是所有权?

Rust的所有权系统是它最核心的特性之一。

### 三大规则

1. 每个值都有一个所有者
1. 同一时间只能有一个所有者
1. 所有者离开作用域,值被丢弃
### 代码示例

rust   fn main() {       let s1 = String::from("hello");       let s2 = s1; // s1的所有权转移       println!("{}", s2);   }
